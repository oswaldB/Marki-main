# relances-detail/index.html - Page relances detail

**Fichier**: `app/static/pages/relances-detail/index.html`

## Description

Page de détail d'une relance avec fonctionnalités IA (ChatGPT et suggestions).

## Workflows

- initial-load.js : Chargement initial
- load-relance.js : Charger les données de la relance
- open-chatgpt.js : Ouvrir le modal ChatGPT
- open-ia-suggestions.js : Ouvrir les suggestions IA
- send-email.js : Envoyer email de relance
- update-status.js : Mettre à jour le statut

## Mockups

- default.html : État normal
- ia-chatgpt.html : Modal ChatGPT ouvert
- ia-suggestions.html : Modal suggestions IA
- empty.html : Erreur - relance non trouvée
- loading.html : Chargement
