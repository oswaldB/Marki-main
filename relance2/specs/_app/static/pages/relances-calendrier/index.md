# relances-calendrier/index.html - Calendrier des relances

**Fichier** : `app/static/pages/relances-calendrier/index.html`

## Description

Vue calendrier des relances programmées. Affiche les relances à venir par date.

## Workflows

- initial-load.js : Chargement initial
- change-month.js : Changer de mois
- select-date.js : Sélectionner une date
- view-relance.js : Voir détail d'une relance depuis le calendrier
- filter-by-type.js : Filtrer par type de relance

## Mockups

- default.html : Vue calendrier normale
- empty-month.html : Aucune relance ce mois
- date-selected.html : Date sélectionnée avec relances
