# impayes-payeur/index.html - Vue par payeur

**Fichier** : `app/static/pages/impayes-payeur/index.html`

## Description

Vue des impayés regroupés par payeur/client. Permet de voir tous les impayés d'un même contact.

## Workflows

- initial-load.js : Chargement initial
- select-payeur.js : Sélectionner un payeur
- view-impayes.js : Voir les impayés du payeur
- create-relance.js : Créer une relance pour ce payeur
- contact-payeur.js : Contacter le payeur
- filter-by-montant.js : Filtrer par montant total

## Mockups

- default.html : Liste des payeurs avec leurs impayés
- payeur-selected.html : Détail d'un payeur sélectionné
- empty.html : Aucun payeur avec impayés
