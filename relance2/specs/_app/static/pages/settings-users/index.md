# settings-users/index.html - Gestion utilisateurs

**Fichier** : `app/static/pages/settings-users/index.html`

## Description

Page de gestion des utilisateurs de l'application (admin uniquement).

## Workflows

- initial-load.js : Chargement des utilisateurs
- create-user.js : Créer un utilisateur
- edit-user.js : Modifier un utilisateur
- delete-user.js : Désactiver un utilisateur
- reset-password.js : Réinitialiser mot de passe
- toggle-role.js : Changer le rôle (admin/user)
- filter-users.js : Filtrer la liste

## Mockups

- default.html : Liste des utilisateurs
- empty.html : Aucun utilisateur
- create-modal.html : Modale création utilisateur
- edit-modal.html : Modale édition utilisateur
