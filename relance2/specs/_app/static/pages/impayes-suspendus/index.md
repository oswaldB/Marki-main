# impayes-suspendus/index.html - Impayés suspendus

**Fichier** : `app/static/pages/impayes-suspendus/index.html`

## Description

Liste des impayés actuellement suspendus (en pause de relance). Permet de gérer les suspensions.

## Workflows

- initial-load.js : Chargement des impayés suspendus
- filter-by-date.js : Filtrer par date de suspension
- unsuspend-impaye.js : Réactiver un impayé
- extend-suspension.js : Prolonger la suspension
- view-detail.js : Voir détail de la suspension
- batch-unsuspend.js : Réactiver plusieurs impayés

## Mockups

- default.html : Liste des impayés suspendus
- empty.html : Aucun impayé suspendu
- extend-modal.html : Modale prolongation suspension
