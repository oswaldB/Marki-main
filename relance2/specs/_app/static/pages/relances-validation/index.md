# relances-validation/index.html - Validation des relances

**Fichier** : `app/static/pages/relances-validation/index.html`

## Description

Page de validation avant envoi des relances. Permet de reviewer et approuver les relances en attente.

## Workflows

- initial-load.js : Chargement des relances à valider
- select-relance.js : Sélectionner une relance
- validate-relance.js : Valider une relance
- reject-relance.js : Rejeter une relance
- batch-validate.js : Valider plusieurs relances
- send-validated.js : Envoyer les relances validées

## Mockups

- default.html : Liste des relances à valider
- all-selected.html : Toutes les relances sélectionnées
- validated.html : Après validation
- empty.html : Aucune relance à valider
