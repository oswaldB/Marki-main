# settings-smtp/index.html - Gestion SMTP

**Fichier** : `app/static/pages/settings-smtp/index.html`

## Description

Page de gestion des profils SMTP pour l'envoi des emails de relance.

## Workflows

- initial-load.js : Chargement des profils SMTP
- create-profil.js : Créer un nouveau profil
- edit-profil.js : Modifier un profil
- delete-profil.js : Supprimer un profil
- test-profil.js : Tester la connexion SMTP
- set-default.js : Définir comme profil par défaut

## Mockups

- default.html : Liste des profils SMTP
- empty.html : Aucun profil configuré
- create-modal.html : Modale création profil
- edit-modal.html : Modale édition profil
