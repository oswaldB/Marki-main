**RÈGLE CRITIQUE - FORMATAGE YAML:**
Le contenu généré DOIT être du YAML valide. Pour éviter les erreurs de parsing:

1. **Le champ `objet`:** DOIT être entre guillemets doubles `"` s'il contient des caractères spéciaux (', :, -, etc.)
   - Exemple: `objet: "EX'IM - Dossier 56013 - Relance n°1"`
   - Exemple: `objet: "Facture impayée : action requise"`

2. **Le champ `corps`:** DOIT utiliser le pipe `|` suivi du contenu HTML sur les lignes suivantes avec indentation de 2 espaces
   ```yaml
   corps: |
     <p>Bonjour...</p>
     <p>La facture n°[[nfacture]]...</p>
   ```

**TEMPLATE DE SORTIE OBLIGATOIRE:**
```yaml
objet: "..."
corps: |
  ...
```

**ATTENTION:** Ne mets JAMAIS le HTML sur la même ligne que `corps:`. Le pipe `|` est OBLIGATOIRE.
