## 📧 Stratégie de Capture & Conversion

**Principe :** Chaque outil demande **email + nom + structure + profil** pour :

1. **Envoyer le résultat** personnalisé (PDF, lien démo, etc.)
2. **Qualifier le lead** (indépendant vs bureau, diag classique vs TANTIUM, particulier vs pro)
3. **Segmenter** la séquence de nurture (par type d'outil utilisé)
4. **Convertir** vers essai gratuit du module correspondant

**Exemple de parcours type :**
```
Utilisateur → Calculateur Tantiums (#2) 
→ Résultat par email avec guide PDF
→ Séquence "TANTIUM" (3 emails éducatifs) 
→ Offre essai gratuit TantiumManager (14 jours)
→ Activation module payant
```
