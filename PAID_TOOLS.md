## 💳 Module de Paiement

### 10. Marki Paiement — Paiement par QR Code

| | |
|---|---|
| **ID** | `marki-paiement` |
| **Cible** | Tous les utilisateurs Marki |
| **Problème résolu** | Impayés, attente de virement, perte de temps de relance |

**Fonctionnalités clés :**
- 📱 QR Code généré automatiquement pour chaque diagnostic
- 🏦 Paiement par Open Banking (connexion directe banque à banque)
- ⚡ Encaissement immédiat (j+1)
- 🔒 Sécurisé (SCA, conforme DSP2)
- 📊 Suivi des encaissements en temps réel
- 🔔 Relance automatique en cas de non-paiement
- 📄 Export comptable automatique

**Modèle économique :**
- **Gratuit** : Pas d'abonnement, uniquement un markup de 1,5% sur les transactions
- **Condition** : Utilisation obligatoire pour bénéficier des modules gratuits
- **Avantage** : Finance l'ensemble de l'écosystème Marki

## 🚀 Démarrage Rapide

```bash
# Cloner le repository
git clone https://github.com/votre-org/marki.git
cd marki

# Lancer un module spécifique
cd rdv
npm install
npm run dev

# Ou activer Marki Paiement (recommandé)
cd marki-paiement
npm install
npm run setup
# → Tous les autres modules deviennent gratuits !
```
